Projeto de Produtos em Java

Programa simples para praticar **herança e polimorfismo** em Java.

## Funcionalidades
- Classe `Product`
- Subclasses `UsedProduct` e `ImportedProduct`
- Sobrescrita de métodos (`priceTag`, `finalPrice`)
- Uso de lista polimórfica para armazenar produtos
- ## Exemplo de saída
PRICE TAGS: Tablet $ 280.00 (Customs fee: $20.00) Notebook $ 1100.00 Iphone (used) $ 400.00 (Manufacture date: 15/03/2017)

TOTAL PRICE: 1780.00
